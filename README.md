# 🏥 Elderly Nursing & Healthcare Assistance Platform

A responsive web-based healthcare platform that connects elderly individuals and their families with verified caregivers, nurses, physiotherapists, and attendants for home-based care services. The platform simplifies caregiver discovery, booking, and service tracking while improving accessibility, safety, and continuity of elderly care.

---

## 📌 Problem Statement

Elderly individuals often require regular medical attention, personal care, and companionship. Traditional elder care services face several challenges:

- Difficulty finding trusted and verified caregivers  
- Lack of transparency in service pricing  
- Manual booking and coordination  
- Inconsistent service quality  
- Limited access to home-based healthcare services  

This platform addresses these issues by providing a centralized digital solution.

---

## 🎯 Objectives

- Digitize elderly nursing and healthcare assistance services  
- Connect families with verified caregivers  
- Enable easy booking and scheduling  
- Improve quality and reliability of care  
- Provide real-time service tracking  

---

## ✨ Features

### 👨‍👩‍👧 User Features
- Secure user registration and login  
- Create elderly/patient profile  
- Browse available healthcare services  
- View caregiver profiles and ratings  
- Book home care services  
- Track booking status  
- View service history  

### 👩‍⚕️ Caregiver Features
- Caregiver registration  
- Manage availability  
- Accept or reject service requests  
- Update service status  
- View work history  

### 🛠 Admin Features
- Verify caregivers  
- Manage users  
- Monitor bookings  
- Handle complaints  
- View analytics dashboard
